Office文档在线编辑有哪些注意事项？

“有时打开文件，提示文件存取错误”：
1、安装的office软件可能是精简版的，重新安装个标准版或者专业版就可以。
2、若是使用360浏览器出现文件存取错误问题，请换IE浏览器使用在线编辑。
3、有可能是IE版本的兼容性视图导致，请添加网站兼容性视图。
4、有可能是IE缓存满了，清一下IE缓存。或者是磁盘的空间不足，查看一下OA系统存放的路径，再看一下该盘中的可用空间是多少，如果少的话就删除一些不必要的文件以释放空间。
5、有可能是ntko控件安装问题
1）首先卸载控件，卸载控件工具下载地址：http://www.tongda2000.com/download/oa_option.php#B3下载NTKO文档在线编辑控件卸载工具，把ntko控件卸载。

2）然后把OA加为受信任站点，然后在IE工具栏中选择internet选项->安全->自定义安全级别，把“ActiveX控件和插件”都调成启用或者提示。点击确定。



3）在http://www.tongda2000.com/download/oa_option.php#B3中下载对应版本的NTKO文档在线编辑控件独立安装包，安装ntko组件。

电子印章:
Office文档在线编辑时可以加盖电子印章，电子印章文件后缀名为.esp，用于提高Office文档安全性和权威性，既可以从客户机选择电子印章加盖，也可以从OA文件柜等处选择加盖。电子印章文件的制作方法，请咨询技术支持人员。
注意：
1、如果其他有编辑权限的用户正在编辑此文件，会提示“某某正在编辑此文件，请稍后重试“，需等待其他人关闭后再打开编辑。
2、用office2007创建的word或excel文件，如果其他人机子上的office是2003版的，不能在线打开这个表格，只能将word的文件保存为doc格式或EXCEL表格保存为XLS格式的。安装2003版office的机器才能打开。
技巧：在系统管理/系统参数设置中，可以设置在线编辑是否保留痕迹和是否显示痕迹。
