在线编辑文档时提示：文件读取错误？

解决方法:

1)一般不会所有的PC都出现读取错误，基本都是某些机子，首先检查c:\windows\DownLoad Program Files 下面有没有NTKO控件（XP）或者OfficeControl.inf OfficeControl.ocx （win7）如果有，查看是否是安装状态，如果是拔出或者其它状态，将其删除掉，去官网下载NTKO控件（10版本以前的版本装4.0，10版本以后装5.0）让后重新登录OA，查看在线阅读是否能正常打开。
2)如果上面的操作后还是读取错误，那么将手工安装的NTKO控件卸载掉，将c:\windows\DownLoad Program Files下的NTKO的控件或者OfficeControl.inf OfficeControl.ocx删除，然后重新登录OA，会自动提示加载NTKO控件，默认的安装在c:\windows\DownLoad Program Files，然后进行在线编辑，这样基本的在线编辑提示文件读取错误都能解决。
