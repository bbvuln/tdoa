重新注册服务的时候一直停留在智能配置上，服务注册不上的原因？

解决方法：

1）检查登录服务器的是不是管理员身份
2）检查注册表中有无注册信息，在运行中输入regedit在HKEY_LOCAL_MACHINE-controlset001-services看一下有没有mysql服务，看看有没有office开头的其它服务。
3）看看MYOA\mysql5\share 这个文件夹是不是存在，如果不存在是注册不了的。
在注册表中注册，注册服务的命令如下：
office_task服务注册
d:/myoa/bin/officetask.exe -name office_task -install office_task
office_im服务注册
d:/myoa/bin/ OfficeIm.exe -name office_im -install office_im
Office_Index服务注册
d:/myoa/bin/ OfficeIndex.exe -name Office_Index Cinstall Office_Index
Office_Daemon服务注册
d:/myoa/bin/ OfficeDaemon.exe -name Office_Daemon Cinstall Office_Daemon
Office_Mail服务注册
d:/myoa/bin/ OfficeMail.exe -name Office_Mail Cinstall Office_Mail
MYSQL5_OA服务注册
D:/MYOA/mysql5/bin/mysqld.exe -install MySQL5_OA
Office_anywhere服务注册
d:/myoa/bin/apache.exe -k install -n Office_Anywhere
