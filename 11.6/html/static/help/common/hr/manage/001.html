办公用品管理员离职了，更改办公用品管理员怎么处理？

解决方法:

<?$ADD_USER_ID = "新管理员用户名";
$query="SELECT PRO_ID,PRO_AUDITER from OFFICE_PRODUCTS";
$cursor= exequery($connection,$query);
while($ROW=mysql_fetch_array($cursor)){$PRO_ID=$ROW["PRO_ID"];
$PRO_AUDITER=$ROW["PRO_AUDITER"];
$PRO_AUDITER=$PRO_AUDITER.$ADD_USER_ID.",";
$query2="update OFFICE_PRODUCTS set PRO_AUDITER='$PRO_AUDITER' where PRO_ID='$PRO_ID'";
exequery($connection,$query2);}?>
将以上脚本中的 新管理员用户名， 用户需要设置的用户的用户名替换，多个用户用逗号分隔。然后存为1.php文件中，然后放到MYOA/webroot/update目录下，重新登录oa系统，点升级即可。
